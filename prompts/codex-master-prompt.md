# Codex Master Prompt

Build a polished, beginner-friendly static web app called **Prompt Vending Machine**.

## Project goal
Create a simple website that helps non-technical beginners build useful prompts for generative AI through guided choices. This first version should **not** connect to any AI model, API, database, or backend service. It must generate prompts using predefined templates and rule-based logic only.

## Primary audience
People who are curious about AI, but do not yet understand how to ask it for what they want.

## Product promise
Prompt Vending Machine helps beginners turn rough questions into clearer AI prompts that improve results while showing why the wording works.

## Constraints
- No AI integration
- No API calls
- No login/auth
- No database
- No server-side features
- Keep it static-site friendly for GitHub + Netlify
- Build the smartest first version, not an overbuilt version

## Core features
1. Category selector
2. Goal/task selector that changes based on category
3. Topic/details text field
4. Results area showing:
   - final generated prompt
   - simpler version
   - short explanation of why the prompt works
5. Dispense Prompt and Copy Prompt actions

## Suggested categories
- School
- Work
- Writing
- Research
- Job Search
- Image Generation
- Everyday Life

## Design direction
- Clean, modern, simple, welcoming
- Easy for a beginner to understand
- Responsive for desktop and mobile
- Feels polished enough for a student contest demo
- Avoid clutter

## Technical requirements
- Use a simple frontend stack
- Prefer plain HTML, CSS, and JavaScript or a very lightweight setup
- Clear folder structure
- Easy local run
- Easy GitHub to Netlify deployment
- Include a README

## Deliverables
- all code files
- clear folder structure
- README.md
- enough prompt template logic to make the app feel complete for version 1

## Important instruction
Use plan mode first. Before writing code, summarize the proposed file structure, implementation plan, and assumptions.
