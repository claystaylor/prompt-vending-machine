# Prompt Vending Machine - Feature List

## Final Core Features

1. **Category selector**  
   Helps the user choose the general area they want help with: School, Work, Writing, Everyday Life, or Image Generation.

2. **Goal/task selector that changes based on category**  
   Helps the user narrow their intent by picking a specific kind of help that fits the category they selected.

3. **Topic/details text field**  
   Lets the user describe what they want in their own words so the tool can build a more useful prompt.

4. **Prompt results area**  
   Shows the Full Prompt, the Simpler Prompt, and a short Why It Works explanation.

5. **Dispense Prompt action**
   Generates a structured prompt from the selected category, selected task, and cleaned topic/details input.

6. **Separate copy buttons**
   Let the user copy either the longer, more complete Full Prompt or the shorter, easier Simpler Prompt.

7. **AI app launch links**
   Appear after the user copies a prompt and let the user open ChatGPT, Claude, or Google Gemini in a new tab.

8. **Input cleanup**
   Trims whitespace, removes awkward trailing punctuation, normalizes simple question phrasing, and removes obvious leading filler phrases before inserting the topic into generated prompts.

9. **Image generation prompt templates**
   Image Generation tasks output completed image prompts with subject, style, mood, composition, details, and exclusions.

## Possible Later Improvements
- Reset button
- Example prompt presets
- Slight prompt variations for ChatGPT, Claude, and Gemini
