# Prompt Vending Machine - Core Feature List

## Version 1 Core Features

1. **Category selector**  
   Helps the user choose the general area they want help with, such as school, work, writing, research, job search, image generation, or everyday life.

2. **Goal/task selector that changes based on category**  
   Helps the user narrow their intent by picking a specific kind of help that fits the category they selected.

3. **Topic/details text field**  
   Lets the user describe what they want in their own words so the tool can build a more useful prompt.

4. **Prompt results area**  
   Shows the final generated prompt, a simpler version, and a short explanation of why the prompt works.

5. **Dispense Prompt and Copy Prompt actions**  
   Let the user generate the prompt and quickly copy it into an AI tool like ChatGPT, Claude, or Gemini.
