# Prompt Vending Machine - Concept Summary

## Project Name
Prompt Vending Machine

## Audience
People who are curious about AI, but do not yet understand how to ask it for what they want.

## Product Promise
Prompt Vending Machine helps beginners turn rough questions into clearer AI prompts that improve results while showing why the wording works.

## What It Is
A beginner-friendly static website that guides users through a few simple choices, then generates a better-formatted prompt they can paste into ChatGPT, Claude, Gemini, or another AI tool.

## What It Is Not
Prompt Vending Machine is not an AI chatbot and does not answer questions itself. It helps users build better prompts for AI tools.

## Version 1 Goal
Build a simple, polished, demo-ready prompt builder that is easy to understand, easy to use, and easy to explain in the contest submission.

## Core User Problem
Beginners often ask AI vague questions and get weak or confusing results because they do not yet know how prompt wording affects output.

## Solution Angle
Help users move from a rough idea to a clearer prompt with guided choices and a short explanation of why the final wording works.

## Platform Plan
- Built with Codex
- Assistance from ChatGPT
- Used in VS Code
- Stored in GitHub
- Deployed with Netlify
- No live AI/API connection in version 1
