# Prompt Vending Machine - Concept Summary

## Project Name
Prompt Vending Machine

## Audience
People who are curious about AI, but do not yet understand how to ask it for what they want.

## Product Promise
Prompt Vending Machine helps beginners turn rough questions, tasks, or ideas into clearer reusable AI prompts while showing why the wording works.

## What It Is
A beginner-friendly static website that guides users through a few simple choices, then generates a Final Prompt, a Simpler Version, and a Why It Works explanation. Users can copy either prompt and open ChatGPT, Claude, Google Gemini, or another AI tool in a new tab.

## What It Is Not
Prompt Vending Machine is not an AI chatbot and does not answer questions itself. It helps users build better prompts for AI tools.

## Version 1 Goal
Build a simple, polished, demo-ready prompt builder that is easy to understand, easy to use, and easy to explain in the contest submission.

## Core User Problem
Beginners often ask AI vague questions and get weak or confusing results because they do not yet know how prompt wording affects output.

## Solution Angle
Help users move from a rough idea to a clearer prompt with guided choices, lightweight topic cleanup, reusable prompt templates, and a short explanation of why the final wording works.

## Final Feature Summary
- Choose a category
- Choose a goal/task
- Enter a topic, question, or details
- Dispense a prompt
- Copy the Final Prompt or Simpler Version
- Open ChatGPT, Claude, or Google Gemini in a new tab
- Read Why It Works to understand the prompt structure

## Platform Plan
- Built with Codex
- Assistance from ChatGPT
- Used in VS Code
- Stored in GitHub
- Deployable as a static site
- No live AI/API connection
